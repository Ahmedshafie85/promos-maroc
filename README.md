# 🇲🇦 Promos Maroc

أفضل العروض والتخفيضات في المغرب — في تطبيق واحد.

Les meilleures promotions du Maroc dans une seule application.

## 🌐 Live

- **Public app:** https://Ahmedshafie85.github.io/promos-maroc/
- **Admin dashboard:** https://Ahmedshafie85.github.io/promos-maroc/admin.html

## ✨ Features

- 🛒 Aggregates deals from major Moroccan retailers
- 🏪 Stores: Marjane, Carrefour, BIM, Aswak Assalam, Atacadão, Label'Vie, Acima, Electroplanet, Biougnach, Decathlon, H&M, Jumia
- 🎟️ Promo codes with one-tap copy
- ❤️ Save favorites (saved in browser)
- 🌍 Arabic (RTL, default) + French toggle
- 📱 Mobile-first responsive design
- ⚙️ Admin dashboard to add/edit/delete deals, stores, categories, promo codes
- 🖼️ Image uploads via Firebase Storage
- 🔐 Email/password admin login
- ⚡ Live updates: changes appear on public site instantly

## 🏗️ Tech stack

- Frontend: Pure HTML/CSS/JS (no framework, no build step)
- Backend: Firebase (Firestore + Storage + Auth)
- Hosting: GitHub Pages
- Cost: Free tier
